PsrLogger

Extends \Psr\Log\AbstractLogger

If you want to use a PSR3 logger, this uses the interface but internally uses com\xcitestudios\Logging\Interfaces\LoggerInterface.

As PSR defines context to be an associative array, some magic will be done to convert this to a sprintf format for message and messageArgs.

If exception exists in context it will be moved as JSON into the extra field.
